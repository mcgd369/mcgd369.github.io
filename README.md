# 澳门指导

澳门水疗场馆专业推荐网站，基于 Next.js 16 构建。

## 快速开始

```bash
npm install --legacy-peer-deps
npm run dev
```

访问 http://localhost:3000

## 构建

```bash
npm run build
```

静态文件输出到 `out/` 目录。

## 部署

推送到 `main` 分支，GitHub Actions 自动构建并部署到 GitHub Pages。

## 技术栈

- Next.js 16 (App Router, Static Export)
- React 19
- Tailwind CSS 4
- shadcn/ui
- TypeScript
