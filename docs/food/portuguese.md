# 葡国菜

澳门的葡国菜是中葡文化交融的完美结晶。不同于传统葡萄牙菜，澳门葡国菜（又称「土生菜」）融合了葡萄牙、中国、印度、马来等多地的烹饪技法和香料，形成了独一无二的风味。

## 招牌葡国菜

### 马介休球（Bacalhau Ball）

**马介休**（Bacalhau）即盐腌鳕鱼，是葡国菜的灵魂食材。据说葡萄牙有超过365种马介休的做法，每天一种不重样。

- **口感**：外酥里嫩，鱼肉丝丝分明
- **推荐搭配**：橄榄油、蒜蓉
- **价格**：约 MOP 30-50/份

::: tip 小知识
马介休在葡萄牙语中意为「鳕鱼」，但澳门人特指盐腌鳕鱼。这种腌制方法源于古代没有冰箱时保存鱼肉的方法。
:::

### 葡式蛋挞（Pastel de Nata）

澳门最具代表性的美食之一，酥脆的挞皮配上嫩滑的蛋奶馅，表面烤得焦香，一口下去满满的幸福感。

### 非洲鸡（Galinha à Africana）

这道菜名字叫「非洲鸡」，实则是澳门独有的创造。用椰奶、咖喱、多种香料调制的酱汁烤制，外皮微焦，肉质鲜嫩，层次丰富。

- **起源**：葡属非洲殖民地的风味，在澳门改良发展
- **特点**：酱汁浓郁，微辣带椰香
- **配饭神器**：一定要配白米饭，酱汁拌饭超香

### 葡国鸡

与非洲鸡不同，葡国鸡的酱汁更浓稠，加入了大量椰奶和咖喱，味道更加温和醇厚。

**主要食材**：
- 鸡腿肉
- 马铃薯
- 洋葱
- 椰奶
- 咖喱粉
- 橄榄油

### 烧牛脷

即烤牛舌，是澳门葡国菜餐厅的常见前菜。牛舌烤得外焦里嫩，配上特制的芥末酱，口感绝妙。

## 平价葡国菜推荐

### 番茄屋葡国美食

- **地址**：半岛清平直街23号
- **人均**：MOP 80-120
- **推荐菜**：马介休球、葡国鸡、木糠布甸
- **特色**：性价比超高的老字号，本地人常去

### 小飞象葡国餐厅

- **地址**：氹仔地堡街喜来登广场
- **人均**：MOP 100-150
- **推荐菜**：非洲鸡、蒜蓉虾、马介休炒饭
- **特色**：官也街附近，游客必打卡

### 法兰度餐厅

- **地址**：路环黑沙海滩附近
- **人均**：MOP 150-200
- **推荐菜**：烤乳猪、马介休、葡国青菜汤
- **特色**：环境优美，海边餐厅

## 葡国菜用餐礼仪

1. **面包是标配**：葡国餐厅通常会先上面包配黄油和橄榄
2. **前菜文化**：建议点几份前菜分享，如马介休球、蒜蓉虾
3. **配酒推荐**：
   - 绿酒（Vinho Verde）：清爽白葡萄酒，配海鲜
   - 波特酒（Porto）：甜型加强酒，配甜品
4. **结账**：服务费通常已包含在账单中，无需额外给小费
