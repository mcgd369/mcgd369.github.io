# 街头小吃

澳门是一座可以「边走边吃」的城市。无论是历史城区的小巷，还是氹仔的官也街，到处都隐藏着令人垂涎的街头小吃。逛吃逛吃，才是打开澳门的正确方式。

## 必吃街头小吃

### 猪扒包

澳门最出名的街头小吃之一，外脆内软的猪仔包夹着一块香煎猪扒，简单却美味。

- **猪仔包**：类似法式面包，外皮酥脆，内里松软
- **猪扒**：用蒜头等香料腌制后煎制，鲜嫩多汁
- **经典吃法**：现点现做，趁热吃最香

| 知名店铺 | 地址 | 价格 |
|---------|------|------|
| 大利来记猪扒包 | 氹仔官也街 | MOP 58/个 |
| 世家猪扒包 | 半岛议事亭前地 | MOP 45/个 |
| 金龙猪扒包 | 半岛新马路 | MOP 38/个 |

::: warning 限量发售
大利来记的猪扒包每天限量，下午2点开始卖，卖完就没了，想吃要趁早！
:::

### 杏仁饼

澳门最具代表性的伴手礼，香脆可口，杏仁香味浓郁。

- **种类**：
  - 原味杏仁饼：经典口味
  - 肉松杏仁饼：咸甜交织
  - 蛋黄杏仁饼：加了咸蛋黄
  - 紫菜杏仁饼：创新口味

**推荐品牌**：
- **钜记手信**：澳门最大的手信品牌，品种最多
- **咀香园饼家**：老字号，传统工艺
- **最香饼家**：本地人推荐，价格实惠

### 肉干

澳门的肉干也是必买伴手礼，现切现烤，香气四溢。

- **口味**：蜜汁猪肉干、黑椒牛肉干、咖喱猪肉干等
- **特点**：大片厚实，肉质鲜嫩，甜咸适中
- **保存**：真空包装可保存约1个月

::: tip 试吃福利
在官也街和议事亭前地，肉干店都提供免费试吃，可以先尝后买，不怕踩雷。
:::

### 鸡蛋仔

港式经典小吃，在澳门也随处可见。外脆内软，蛋香浓郁。

- **经典口味**：原味、巧克力、抹茶
- **创新口味**：芝士、奥利奥、珍珠奶茶味

### 牛杂

澳门的牛杂很有特色，用各种香料卤制，配上咖喱酱或甜酱，是街头巷尾的人气小吃。

- **部位**：牛肚、牛肠、牛肺、萝卜
- **酱料**：咖喱酱、甜酱、辣椒酱
- **推荐**：恒友鱼蛋牛杂（半岛大堂巷）

## 官也街美食地图

官也街（Rua do Cunha）位于氹仔，是澳门最有名的美食街，短短100多米的街道上聚集了数十家美食店。

### 必吃清单

1. **世记咖啡** → 瓦煲咖啡、猪扒包
2. **莫义记** → 猫山王榴莲雪糕、芒果大菜糕
3. **大利来记** → 猪扒包
4. **安德鲁饼店** → 葡挞
5. **诚昌饭店** → 水蟹粥
6. **钜记手信** → 杏仁饼、肉干
7. **晃记饼家** → 老婆饼、杏仁饼

::: info 怎么去官也街
- 从澳门半岛：乘公交车11、22、28A到「氹仔官也街」站
- 从路氹城：步行即可，距离威尼斯人约10分钟
:::

## 议事亭前地周边美食

议事亭前地是澳门半岛的中心，周边小巷里藏着很多美食。

### 推荐路线

```
议事亭前地 → 大堂巷 → 板樟堂 → 营地大街 → 十月初五街
```

## 路环小镇美食

路环是澳门的「后花园」，这里更安静，更有葡式风情。

### 安德鲁饼店总店

- **地址**：路环市中心斜巷1号
- **招牌**：葡式蛋挞、面包
- **特色**：玛嘉烈蛋挞的「娘家」，环境悠闲

### 法兰度餐厅

- **地址**：路环黑沙海滩
- **招牌**：烤乳猪、马介休
- **特色**：海边葡国餐厅

### 汉记手打咖啡

- **地址**：路环码头
- **招牌**：手打咖啡、西多士
- **特色**：400次手打咖啡，香浓顺滑
