# 美食地图

澳门虽然不大，但美食遍布各个区域。这张美食地图将带你按区域探索澳门的美味，从半岛到路环，每一处都有惊喜。

## 澳门半岛

澳门半岛是澳门的历史中心，也是老字号美食最集中的区域。

### 议事亭前地周边

澳门最热闹的商业区，周边小巷藏着无数美食。

**必吃清单：**
- 🥛 **义顺牛奶公司** → 双皮奶、姜撞奶
- 🍦 **柠檬车露** → 手工冰淇淋
- 🍜 **恒友鱼蛋** → 咖喱牛杂、鱼蛋
- 🥟 **六记粥面** → 鲜虾云吞面
- 🍮 **荣记豆腐** → 豆腐花

### 大三巴周边

大三巴是澳门地标，周边也有很多美食。

**必吃清单：**
- 🥧 **玛嘉烈蛋挞** → 葡式蛋挞
- 🍞 **世家猪扒包** → 猪扒包
- 🍵 **龙华茶楼** → 传统早茶
- 🍲 **番茄屋葡国美食** → 平价葡国菜

### 十月初五街 / 福隆新街

老街巷里的美食，充满怀旧气息。

**必吃清单：**
- 🍵 **六国饭店** → 怀旧西餐厅
- 🥟 **大龙凤茶楼** → 传统粤式点心
- 🍝 **祥记面家** → 虾子捞面

---

## 氹仔

氹仔是澳门的「美食后花园」，官也街更是美食一条街。

### 官也街

澳门最有名的美食街，短短的街道上聚集了数十家美食店。

**必吃清单：**
- 🐷 **大利来记猪扒包** → 猪扒包
- 🍮 **莫义记** → 榴莲雪糕、大菜糕
- 🍪 **晃记饼家** → 老婆饼、杏仁饼
- 🍳 **世记咖啡** → 瓦煲咖啡、猪扒包
- 🍲 **诚昌饭店** → 水蟹粥
- 🥧 **安德鲁饼店** → 葡挞

### 地堡街 / 飞能便度街

官也街的延伸，也有很多好吃的。

**必吃清单：**
- 🐘 **小飞象葡国餐厅** → 葡国菜
- 🍮 **沙度娜** → 木糠布甸
- 🦞 **山渡士葡国餐厅** → 葡国海鲜

---

## 路环

路环是澳门最安静的区域，充满葡式小镇风情。

### 路环市区

- 🥧 **安德鲁饼店总店** → 葡挞的发源地
- 🐔 **安德鲁花园咖啡** → 葡式简餐
- ☕ **汉记手打咖啡** → 400次手打咖啡

### 黑沙海滩

- 🍖 **法兰度餐厅** → 海边葡国菜，烤乳猪是招牌
- 🏖️ **黑沙海滩烧烤** → 假日沙滩烧烤

- **永利美食广场**：品质不错

## 美食路线推荐

### 经典一日游路线

```
早上：半岛喝早茶（龙华茶楼）
   ↓
上午：逛大三巴 + 玛嘉烈蛋挞
   ↓
中午：葡国菜午餐（番茄屋）
   ↓
下午：氹仔官也街逛吃
   ↓
傍晚：路环安德鲁 + 海边散步
   ↓
晚上：返半岛吃宵夜
```

### 小吃路线

```
议事亭前地（义顺双皮奶）
   → 大堂巷（柠檬车露 + 恒友鱼蛋）
   → 板樟堂（猪扒包）
   → 氹仔官也街（各种小吃一路吃）
```

### 葡国菜路线

```
午餐：番茄屋（平价葡国菜）
   → 下午茶：安德鲁蛋挞
   → 晚餐：小飞象 / 法兰度（正餐葡国菜）
   → 甜品：木糠布甸
```

## 交通指南

### 澳门半岛 ↔ 氹仔/路环

- **公交车**：MOP 6/程，最经济实惠
- **的士**：约 MOP 50-80
- **赌场免费巴士**：各大酒店都有免费穿梭巴士
- **轻轨**：氹仔线已开通，连接口岸和氹仔

### 步行推荐

半岛中区和氹仔官也街都适合步行逛吃，景点和美食都很集中。

::: tip 交通小贴士
1. 赌场巴士（发财车）是免费的，可以利用
2. 澳门不大，很多地方走路就能到
3. 出租车可以打电话叫车：28812345
:::

## 实用信息

### 营业时间

- **早餐/早茶**：6:30 - 11:00
- **午餐**：11:30 - 14:30
- **下午茶**：14:30 - 17:30
- **晚餐**：18:00 - 22:00
- **宵夜**：22:00 - 凌晨

### 支付方式

- 大部分餐厅接受**支付宝、微信支付**
- 建议备一些**澳门元现金**，小摊小店可能只收现金
- **港币**也通用，但通常按1:1使用，不太划算

### 人均消费参考

| 类型 | 人均（MOP） |
|------|------------|
| 街头小吃 | 30-60 |
| 茶楼点心 | 80-150 |
| 平价葡国菜 | 100-200 |
| 中档餐厅 | 200-500 |
| 米其林餐厅 | 800-2000+ |
